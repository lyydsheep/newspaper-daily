Sysdig 澄清首例"智能体勒索软件"JadePuffer：AI 执行攻击但人类仍负责设置与选目标
云安全公司Sysdig记录了首例由AI智能体独立执行的勒索软件攻击JadePuffer，AI完成了从入侵到加密的完整攻击链，但人类仍负责设置基础设施和选择目标。这打破了"完全自主AI攻击"的叙事，提醒业界AI是攻击效率的倍增器，而非攻击的起源。
Gemini Spark 可智能追踪话题实时反应
Gemini Spark新增话题追踪与实时事件反应能力，用户可在比赛结束后获取定制化分析邮件。这标志着AI助手从被动问答向主动事件驱动的转变。
OfficeCLI：为AI智能体设计的开源Office套件
OfficeCLI是全球首个专为AI智能体设计的开源Office套件，以单二进制文件运行，无需安装Office。它通过"渲染→查看→修复"的视觉闭环，使AI代理能自主创建和修改Word、Excel、PowerPoint文档，支持公式、图表等复杂功能。
Claude Code v2.1.202 发布
Claude Code v2.1.202新增动态工作流规模设置，可控制agent数量规模，并修复了mTLS握手失败、移动端图片发送等多项问题。持续优化AI编程助手的稳定性和灵活性。
Claude Code 团队详解四种智能体循环类型
Claude Code团队将智能体工作循环分为回合循环、目标循环、时间循环和主动循环四种类型，每种对应不同的触发方式和停止条件。这为开发者设计AI工作流提供了清晰的框架指导。
2026年科技公司AI裁员名单：Microsoft、Oracle、GitLab等十家公司裁减数千岗位
2026年以来，至少十家科技公司以AI为由大规模裁员，包括Microsoft裁减4800人、Oracle裁减21000人、Meta裁减8000人等。AI正在重塑科技行业的就业结构，但裁员与AI投资之间的直接因果关系仍需审慎看待。
Claude Fable实地指南：发现你的未知
Claude Fable是首款要求用户主动澄清未知才能获得高质量工作的模型。通过将问题分解为已知的已知、已知的未知等四类，用户可借助盲点检查、头脑风暴等方法高效挖掘代码库中的潜在问题。
OpenClaw 登陆 HuggingFace 本地应用
OpenClaw现已登陆HuggingFace本地应用，用户可选择任意GGUF/MLX模型，快速部署完全本地的工具调用智能体，无需云端、无需密钥、无需监控。
SGLang 集成 DSpark 推测解码：置信度驱动的可变长度验证
SGLang团队将DSpark推测解码算法集成到开源推理引擎中，通过置信度头和顺序温度缩放动态分配可变验证长度，在高负载下裁剪无效验证成本。支持密集和稀疏模型，提供三种验证模式。
Google 更新隐私设置，默认用媒体数据训练 AI，用户可手动退出
Google于6月低调更新搜索服务隐私设置，默认将用户上传的媒体数据用于训练AI模型。用户可通过取消勾选"保存媒体"框退出，并可设置自动删除周期。Meta等公司也在大规模收集用户数据。
