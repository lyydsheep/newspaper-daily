GitHub 开源 Spec Kit 工具包，用产品规范引导 AI 编码
GitHub 开源了 Spec Kit 工具包，旨在通过产品规范文档来引导 AI 编码，提升代码生成的一致性和准确性。该工具将需求规格与代码生成流程紧密结合，有望改变开发者与 AI 协作的方式，减少理解偏差。
OpenCV 5 发布：升级全新 DNN 引擎、原生支持大模型
OpenCV 5 正式发布，带来了全新的深度神经网络（DNN）引擎，并原生支持大语言模型等现代AI架构。这一升级意味着计算机视觉库将能更高效地运行Transformer等复杂模型，为边缘计算和实时应用提供更强动力。
Persona Atlas：Hugging Face 上的开源人物思维映射工具
Persona Atlas 是一个在 Hugging Face 上开源的工具，用于创建和映射不同人物角色的思维模式。它可以帮助开发者设计更逼真的AI角色，或在游戏、模拟器中构建复杂的多智能体交互，拓展了AI在创意领域的应用边界。
美国众议院议员发布法案草案，旨在禁止各州制定人工智能相关法规
美国众议院议员发布了一项法案草案，试图禁止各州自行制定人工智能监管法规，将监管权统一收归联邦。此举旨在避免各州法规碎片化影响行业发展，但也引发了关于消费者保护和地方自主权的激烈争论。

谷歌与SpaceX达成了一项巨额协议，每月支付9.2亿美元以租用xAI数据中心的计算能力。这笔交易凸显了AI算力需求的爆炸性增长，以及科技巨头为争夺稀缺资源不惜砸下重金的激烈竞争。
