DiffusionGemma：文本生成速度提升4倍的开源扩散模型
Google DeepMind发布DiffusionGemma，一种基于扩散技术的文本生成模型，速度比传统自回归模型快4倍。该模型采用非自回归架构，可并行生成文本，在保持质量的同时大幅降低延迟，有望改变大模型推理效率格局。
Grok Voice性能出色价格低廉
xAI宣布Grok语音功能性能优异且定价低廉，进一步降低AI语音交互门槛。此举将加剧与OpenAI、Google等厂商在语音AI领域的竞争，推动语音助手市场进入价格战阶段。
小米发布 MiMo Code V0.1 开源终端 AI 编程助手
小米旗下MiMo团队开源终端AI编程助手MiMo Code V0.1，支持代码生成、补全与调试。这是小米在AI开发者工具领域的重要布局，有望借助开源生态吸引更多开发者，与Cursor、GitHub Copilot等竞争。
MiniMax M3 上链 0G，限时免费运行
MiniMax将其M3模型部署至去中心化存储网络0G，并推出限时免费使用活动。此举探索了AI模型与区块链结合的新模式，可能降低模型部署成本并增强数据主权。
Google将保存用户的Lens图片、Search Live录音和Translate音频用于AI训练
Google更新隐私政策，将默认保存用户通过Lens搜索的图片、Search Live的录音以及Translate的音频数据用于AI训练。此举引发隐私担忧，用户需手动关闭相关设置，凸显科技巨头在数据收集与用户权益之间的持续拉锯。
Cursor Bugbot 更新：速度提升超 3 倍、成本降低 22%、发现更多 Bug
AI编程工具Cursor的Bugbot功能迎来重大更新，自动检测bug速度提升3倍以上，成本降低22%，且能发现更多类型缺陷。这标志着AI辅助代码质量保障进入实用化新阶段，有望大幅减少人工调试时间。
Apache Burr：构建可靠的人工智能代理和应用程序
Apache软件基金会推出Burr框架，旨在帮助开发者构建可靠、可观测的AI代理和应用程序。作为Apache顶级项目，Burr提供了状态管理、错误恢复等关键能力，填补了AI应用工程化基础设施的空白。
华为云发布全球首个端到端具身AI平台CloudRobo
华为云推出CloudRobo平台，号称全球首个端到端具身AI平台，覆盖从感知、决策到执行的完整机器人开发链路。此举将加速机器人与AI融合，推动智能制造、服务机器人等场景落地。
火山方舟版权商业化平台上线，周星驰比高集团三大电影IP首批入驻
字节跳动旗下火山引擎推出方舟版权商业化平台，周星驰旗下比高集团的《功夫》《喜剧之王》等三大电影IP首批入驻。该平台旨在为AI生成内容提供合法版权授权渠道，探索IP商业化新路径。
OpenRouter 推出 Activity explorer 活动探索器
AI模型聚合平台OpenRouter上线Activity Explorer功能，让用户可视化追踪和探索API调用活动。该工具提升了模型使用透明度，便于开发者优化成本和性能。
