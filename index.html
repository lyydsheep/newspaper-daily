PixVerse融资4.39亿美元，估值破20亿

视频生成赛道正在上演一场资本与技术的双重狂欢。PixVerse以4.39亿美元C轮融资和超20亿美元估值，再次印证了AI视频生成领域的巨大想象空间。然而，当我们看到其消费端月活超1500万、图生视频每分钟4.80美元的定价时，不禁要问：这究竟是技术突破带来的价值兑现，还是资本泡沫下的又一注水？更值得警惕的是，PixVerse的R系列世界模型瞄准游戏开发，这标志着AI生成内容正从短视频向高价值、高复杂度的专业领域渗透。腾讯混元HyOCR-1.5的端到端OCR大模型全栈开源，则展示了另一种路径：通过开放技术生态，降低行业门槛，推动AI普惠。在资本与开源的博弈中，谁更能代表AI的未来？答案或许不在估值数字里，而在那些真正改变工作流、降低创作成本的技术细节中。

视频生成创企 PixVerse 完成 4.39 亿美元 C 轮扩展融资，估值超 20 亿美元
新加坡视频生成初创公司 PixVerse 完成 4.39 亿美元 C 轮扩展融资，估值突破 20 亿美元，投资方包括阿里巴巴。公司提供面向消费者、专业影视和游戏开发的三大产品系列，支持 4K 分辨率视频生成，消费端月活超 1500 万。此次融资表明资本市场对 AI 视频生成赛道持续看好，但高估值也引发对商业化可持续性的讨论。
前沿模型实际成本：tokenizer 差异导致隐性涨价
同一份 TypeScript 文件在不同模型的 tokenizer 下 token 数差异达 1.73 倍，Anthropic 新 tokenizer 比旧版多产生约 30% 的 token，标价不变构成隐性涨价。Claude Opus 4.6 与 4.8 标价相同，但新 tokenizer 使代码 token 数增加约 32%。这揭示了模型定价中的隐藏成本陷阱，用户实际支出可能远超预期。
OpenAI 面向普通用户发布提示词指南：从结果出发，少写步骤
OpenAI 发布面向普通用户的提示词指南，建议以结果而非步骤开头，用一两条硬性规则替代逐步骤脚本。指南涵盖目标、上下文、输出格式和边界四个可选模块，并介绍基于 Codex 技术和 GPT-5.6 模型的 ChatGPT Work 处理复杂项目。这表明 AI 交互正从“编程式指令”向“目标导向对话”演进，降低了使用门槛。
德国AI协会发布开源模型Soofi S，在英语和德语基准测试中领先
德国AI协会发布开源大语言模型 Soofi S 30B-A3B，总参数量 316 亿，采用 Mamba-2 与标准注意力层混合的 MoE 架构，在英语和德语基准测试中超越 OLMo 3 32B 等开源模型。该模型完全在德国本土云上训练，德语数据占比逐步提升，体现了欧洲在开源 AI 领域的自主可控努力。
腾讯混元发布 HyOCR-1.5：端到端 OCR 大模型全栈开源，推理提速 6.37 倍
腾讯混元发布 HyOCR-1.5，这是端到端 OCR 大模型领域首个完整开源训练、推理和权重的专家模型。仅 1B 参数，支持 4K 分辨率与 128K 上下文，引入 DFlash 投机解码框架实现 6.37 倍推理加速。该模型扩展了低资源 OCR 和古文字识别能力，有望推动 OCR 技术在更多场景落地。
Meta 宣布扩建路易斯安那州数据中心至 5GW，总投资超 500 亿美元
Meta 将其路易斯安那州数据中心算力扩至 5GW，总投资超 500 亿美元，为全球最大 AI 基础设施投资之一。Meta 承诺承担全部能源及水资源费用，并投资超 10 亿美元改善当地基础设施，同时与能源公司合作支持天然气发电、储能和核电项目。这反映了科技巨头对 AI 算力的巨大需求及对能源基础设施的深度介入。
Seedream 5.0 Pro 测评：图像编辑门槛爆降
字节跳动发布 Seedream 5.0 Pro，图像质量与提示词理解追平 GPT-Image 2.0，核心亮点是“可编辑”交互，支持在图上打点、画框、涂鸦实现精准局部编辑。实测案例涵盖家装改造、商品图制作等场景，火山引擎已全量上线 API。这大幅降低了图像编辑的门槛，让非专业用户也能轻松完成复杂编辑任务。
黄仁勋：英伟达季度收入逼近千亿美元，Rubin Ultra 架构未延期
英伟达 CEO 黄仁勋表示季度营收即将逼近 1000 亿美元，否认下一代旗舰架构 Rubin Ultra 延期传闻，称其按计划明年出货。英伟达预计本财年 CPU 业务收入约 200 亿美元，下一代 Vera CPU 将进军通用服务器市场。这凸显了英伟达在 AI 硬件领域的绝对主导地位及对未来增长的强烈信心。
xAI 官方 Grok CLI 被曝静默上传整个代码库及用户密钥
安全研究者发现 xAI 官方 Grok CLI 会在每轮任务前后静默上传整个工作目录，包括仓库外的敏感配置文件如 ~/.claude.json 和全局 AGENTS 规则。即使模型仅回复一个单词，上传依然发生。这暴露了 AI 开发工具在隐私和安全方面的严重隐患，引发对代码托管和用户数据保护的担忧。
Ploy 将 AI 智能体默认模型从 Claude Opus 4.8 切换至 GPT-5.6 Sol
Ploy 将其 AI 智能体默认模型从 Claude Opus 4.8 切换至 OpenAI 新发布的 GPT-5.6 Sol。在真实营销网站构建测试中，新模型完成页面平均耗时快 2.2 倍，成本降低 27%，视觉评分提升。但迁移过程发现 GPT-5.6 会为所有工具参数填充默认值，导致一些意外行为。这显示了模型切换带来的性能提升与适配挑战并存。
