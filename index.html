GitHub 开源 Spec Kit 工具包，用产品规范引导 AI 编码
GitHub 上开源了一个名为 Spec Kit 的工具包，旨在通过定义产品规范来引导 AI 生成更符合预期的代码。这一工具试图解决当前 AI 编码中“生成结果不可控”的核心痛点，让开发者能够像写需求文档一样指挥 AI 编程，有望提升代码质量和开发效率。
OpenCV 5 发布：升级全新 DNN 引擎、原生支持大模型
计算机视觉库 OpenCV 迎来重大版本更新，5.0 版本引入了全新的深度神经网络（DNN）引擎，并原生支持大语言模型。这意味着开发者可以直接在 OpenCV 中调用和部署大模型，无需额外集成其他框架，将极大简化视觉AI应用的开发流程。
Persona Atlas：Hugging Face 上的开源人物思维映射工具
Hugging Face 社区发布了一款名为 Persona Atlas 的开源工具，用于构建和映射人物的思维模式与人格特征。该工具可应用于角色扮演游戏、虚拟助手个性化等场景，为AI系统赋予更丰富、更一致的人物设定，是AI人格化探索的有趣尝试。
美国众议院议员发布法案草案，旨在禁止各州制定人工智能相关法规
美国众议院议员公布了一项法案草案，意图禁止各州自行制定人工智能法规，将监管权统一收归联邦。此举旨在避免各州法规碎片化给科技企业带来的合规负担，但也引发了关于联邦监管是否足够严格、是否会削弱地方创新活力的争议。
谷歌将每月向SpaceX支付9.2亿美元，以获取xAI数据中心的计算能力
据CNBC报道，谷歌与SpaceX达成一项巨额协议，每月支付9.2亿美元以获取xAI数据中心的计算能力。这笔交易凸显了AI大模型训练对算力的极度渴求，以及科技巨头间围绕稀缺计算资源展开的激烈争夺，算力已成为AI时代最昂贵的“硬通货”。
