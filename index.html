全球首例 AI Agent 勒索攻击曝光，从漏洞利用到数据库加密全程自主完成
安全厂商 Sysdig 首次记录到 AI Agent"JADEPUFFER"自动完成的勒索攻击。攻击利用暴露的 Langflow 服务漏洞远程执行代码，随后自主收集多家AI和云平台凭证，横向移动到数据库并加密。这标志着AI Agent已具备完整的攻击链自主执行能力，对网络安全防御提出全新挑战。
生数科技发布 Vidu S1，推动视频生成迈向"实时交互"新时代
7月3日，生数科技发布Vidu S1实时交互模型，支持实时视频通话和语音控制视频走向，实现无限时长连续互动。模型采用自回归扩散路线，在540P分辨率下实现25-42FPS实时生成。该技术将视频生成从单向输出推向双向交互，为虚拟角色、远程协作等场景打开新可能。
pxpipe：通过图像化压缩输入token降低Claude Code成本
pxpipe是一个本地代理，将密集文本渲染为PNG图像，利用图像token成本取决于像素尺寸的特性压缩输入token。在Fable 5模型上，约25k文本token压缩为约2.7k图像token，端到端账单降低59-70%。该方法虽为有损压缩，但在SWE-bench测试中成本大幅下降且结果一致，为高token消耗场景提供了实用降本方案。
Fable 的判断力：Simon Willison 从 Claude Code 团队获得的效率技巧
Simon Willison建议让AI模型用自己的判断力工作，而非硬性规定行为。例如直接让模型自行决定何时编写测试，效果更好。另一技巧是让模型将较小任务委托给较低功耗模型，主循环保留关键任务。这反映了AI协作的最佳实践正从“指令式”转向“委托式”。
面壁智能发布AI全自动预训练框架ForgeTrain，8小时追平Megatron-LM
面壁智能发布全球首个完全由AI编写、无人类干预的生产级大模型预训练框架ForgeTrain。该框架针对特定模型和硬件从零自动生成训练代码，8小时追平Megatron-LM，1.5至2天内实现反超，FLOPS利用率提升约8%-10%。这标志着AI在基础科研工具开发上迈出自主化关键一步。
国家网信办就《互联网信息服务管理办法》再次征求意见，首设"智能信息服务"专章规范AI服务
7月3日，国家网信办发布《互联网信息服务管理办法（修订草案征求意见稿）》，新增"智能信息服务"专章。要求AI服务提供者公示技术原理、训练数据来源，对生成内容进行标识，禁止强制用户使用智能服务或利用算法扰乱舆论。这是中国在AI监管领域的重要立法进展，意见反馈截止8月2日。
面向 Web 开发者的 Safari MCP 服务器
Safari Technology Preview 247推出Safari MCP服务器，基于Model Context Protocol，允许智能体连接Safari浏览器窗口。智能体可获取DOM、网络请求、截图、控制台输出等信息，自主完成调试、性能分析等任务。这为Web开发自动化提供了新的基础设施，有望改变前端开发工作流。
JoyAI App 上线 UGC 数字人功能，用户可"捏"出专属虚拟玩伴
JoyAI App上线UGC数字人功能，用户上传照片即可生成虚拟数字分身，支持写实或卡通风格，搭配用户语音实现全双工对话。该功能集成语言、语音、数字人大模型，可提供点外卖、金融咨询等生活服务。数字人正从娱乐工具向全能助手演进。
藏师傅PPT与Pencil结合使用技巧
用户分享将AI生成PPT导入Pencil设计软件的工作流，可在Pencil中手动调整AI常见的排版问题。Pencil提供更强的编辑能力，支持导出网页和PNG。这种结合大幅提升了AI生成内容的可编辑性，弥补了AI在精细排版上的不足。
Wan Video 推出"音乐伴舞"新功能
Wan Video推出"音乐伴舞"功能，用户上传角色和歌曲后，可生成与节奏同步的舞蹈视频，支持街舞、踢踏舞、拉丁舞等多种舞种。该功能将AI视频生成从静态内容推向动态表演，为创意内容生产提供新工具。
